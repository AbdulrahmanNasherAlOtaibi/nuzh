## 🎯 الفكرة

**نُزهة** تربط عشاق الطبيعة بالمحميات الملكية عبر رحلات بيئية منظمة ومرخصة من شركات سياحية
ومرشدين معتمدين، مع خريطة تفاعلية توضح مناطق التنزه (مسموح / تصريح / ممنوع)، ومحتوى توعوي
يرسخ السلوك البيئي الصحيح.

## ✨ المنصة تشمل ثلاث واجهات

### 1) تطبيق الضيف (عربي RTL — ثيم داكن/فاتح — عربي/إنجليزي)
- **الرئيسية**: كاروسيل إعلاني متحرك، التصنيفات الخمسة (بيئية، ثقافية، ترفيهية، مغامرات، مع مرشد سياحي)، خانة التوعية البيئية (المقالات، اللقطات، تقارير المحميات، السلوك البيئي الصحيح)، أقسام «رحلات يحبها الضيوف» و«أعلى الرحلات تقييماً» و«عروض نهاية الأسبوع»، وشركاء النجاح.
- **استكشف**: شبكة التصنيفات + التصاريح، بحث وفرز.
- **الخريطة**: خريطة تفاعلية للمحميتين بمناطق ملونة (أخضر مسموح، أصفر تصريح، أحمر ممنوع، أزرق موقعي) — الضغط على أي منطقة يظهر تصنيفها فوراً.
- **رحلاتي**: القادمة (+ خطط رحلة جديدة)، السابقة (إرفاق صور + تقييم + مشاركة)، الإحصائيات (إجمالي الرحلات، المسافة، الرحلة المفضلة، تقييم الحساب من 5 نجوم)، الإنجازات (16 إنجازاً)، والتصاريح.
- **حسابي**: الملف الشخصي، المظهر (داكن/فاتح/حسب الجوال)، اللغة، التواصل، عن نزهة، الأسئلة الشائعة، دليل المستخدم، إنجازات المنصة.
- **الحجز**: اختيار التاريخ وعدد الأشخاص وملاحظات وكود خصم وطريقة الدفع (مدى/Visa/Apple Pay).
- ✅ **التقييم موثوق**: لا يستطيع أحد تقييم رحلة إلا إذا حجزها وأكملها فعلياً عبر المنصة.

### 2) بوابة مزودي الخدمة `/provider`
تسجيل كشركة سياحية (اسم الشركة) أو مرشد سياحي (الاسم الرباعي)، إدارة الرحلات
(تُنشر بعد اعتماد الإدارة)، متابعة الحجوزات والإيرادات، والرد على التقييمات.

### 3) لوحة تحكم الأدمن `/admin` — عشرة أقسام
لوحة المعلومات (6 مؤشرات + رسوم بيانية) · إدارة الشركات · إدارة المستخدمين · إدارة الحجوزات ·
الإيرادات والمالية · التقييمات والشكاوى · المحتوى والرحلات · الترويجات والعروض ·
التقارير والتحليلات (تصدير CSV) · الإعدادات والأمان (موظفون، سجلات دخول، نسخ احتياطي).

## 🚀 التشغيل

```bash
npm install
npm run build     # بناء الواجهة
npm start         # التشغيل على http://localhost:5001
```

> للتطوير المباشر: `npm run dev` (خادم API على 5001 + Vite على 5173).

**من الجوال على نفس الشبكة**: افتح `http://<IP جهازك>:5001` — الخادم يستمع على كل الواجهات.

## 🧩 التقنية

| الطبقة | التقنية |
|---|---|
| الواجهة | React 19 + Vite + Tailwind CSS + wouter |
| الخريطة | Leaflet + OpenStreetMap |
| الخادم | Express + tsx (TypeScript) |
| قاعدة البيانات | SQLite (better-sqlite3) — ملف واحد `data/nuzha.db` بدون أي إعداد |
| الخط | Cairo (مستضاف محلياً) |

- تُنشأ الجداول وتُحمّل **بيانات تجريبية غنية** تلقائياً عند أول تشغيل
  (رحلات، حجوزات على 12 شهراً، تقييمات، شكاوى، تصاريح، ترويجات…).
- لإعادة تهيئة البيانات: احذف مجلد `data/` وأعد التشغيل.
- ملاحظة: بلاطات الخريطة تحتاج اتصالاً بالإنترنت.

## 📁 البنية

```
server/    Express + SQLite (db, seed, auth, api, admin)
src/       واجهة React (pages, components, admin, lib)
public/    الشعار والمشاهد الصحراوية (SVG)
```
